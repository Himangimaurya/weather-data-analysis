# Weather Data Analysis

This project analyzes a weather dataset using Python, Pandas, Matplotlib, and Seaborn.

## Tasks Performed
- Loaded CSV data using Pandas
- Calculated average values of numeric columns (Temperature, Humidity, etc.)
- Created a Bar Chart (Average Temperature by Weather Condition)
- Created a Scatter Plot (Temperature vs Humidity)
- Created a Heatmap (Correlation between weather variables)
- Provided insights and observations based on the analysis

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Files
- `weather_analysis.ipynb` - Main notebook with code, charts & insights
- `weatherHistory.csv` - Dataset used for analysis

## Dataset Source
[Szeged Weather Dataset - Kaggle](https://www.kaggle.com/datasets/budincsevity/szeged-weather)
